# Oculus

**Adjustable FOV for Minecraft Beta 1.7.3.**

Oculus lets you set a base FOV and temporarily zoom in using the mouse wheel, giving you precise visual control without altering gameplay.

## Features

- Set a custom FOV preset
- Temporary zoom with mouse wheel
- Standalone and lightweight
- Minimal performance impact
- Era-faithful implementation
